About Our Q&A Chatbot

Powered by BERT and Natural Questions

Technical Details

Our question-answering chatbot was developed using:

  • Model: deepset/bert-base-cased-squad2
  • Training Data: Natural Questions Short dataset
  • Performance Metrics:
    • F1 Score: 80.11%
    • Precision: 77.50%
    • Recall: 88.57%

How It Works

The system uses a fine-tuned BERT model to understand questions and find accurate answers within the provided context. The model was specifically trained on question-answer pairs to excel at extractive QA tasks.

Technology Stack

The backend is built with Flask framework, using PyTorch for the machine learning components and Hugging Face's Transformers library for the BERT implementation.
